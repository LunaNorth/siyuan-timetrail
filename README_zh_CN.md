# 时迹

思源笔记LifeLog数据可视化插件。

## 使用前提

本插件需要配合叶归插件的**LifeLog功能**使用。请确保：

1. 已安装并启用叶归插件
2. 在叶归插件设置中开启LifeLog功能


## 使用方法

1. **启用LifeLog**：在叶归插件设置中启用LifeLog功能
2. **记录时间**：使用叶归插件的快捷键或菜单记录当前时间
3. **查看记录**：打开时迹插件，即可查看相应的LifeLog记录
4. **筛选统计**：按时间范围或类型筛选，查看统计信息

## 功能特性

- 时间轴展示LifeLog记录
- 按时间和类型筛选
- 统计数据可视化
